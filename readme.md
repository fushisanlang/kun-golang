### 基本玩法说明

基本玩法其实就是没有玩法。

开局选方位和属性。东南西北，金木水火土日月。用来确定二十八星宿之一。二十八星宿算是最开始的一个级别。

之后就是二级进化，基本上是一些常见神兽凶兽这样。再加上天干地支的古称。

三级进化就只有特定某些路线有了，名称的话基本是帝君，古神族的姓氏加上日月的古称。相应的进化路线在这。[进化路线图](进化路线图.svg)



进化条件是经验的增加。百经验升级，百级进化。某些不存在三级进化的，在三级进化的时候名称不会变。

~~名称这里其实只是一个中二感的满足，意义不大~~



经验来源主要两点：

* 一个是每秒增加1点。因为升级需要的经验一直是100，所以自动经验获取也不变。
* 另一个是战斗获得，战斗结算如下：
  * 获胜的话，会吸收对方经验的百分之一。算是圆一个吞噬的说法。
  * 失败的话会掉经验，对手的级别也分为一级二级三级。失败了就掉相应的等级。~~这里后续可以改成，掉自己的百分之一，跟上边配合。~~
  * 战斗前会有胜率计算，暂时只基于五行属性。**后续怎么发展，应该是不会发展了，因为不打算在这个项目耗费太多时间了。** 如果胜率小可以选择逃跑，逃跑会掉50点经验。当前经验不够扣就掉级，连锁反应。



游戏结局，或者经验扣光死亡，或者达到四次进化，化鲲为鹏，就算是胜利。



**基本是个挂机加赌狗游戏，因为没有任何认为干预的手段。当然一直狗满级也是可以。**

**这里没设置太多正向激励机制，如果可能，可以加入成就系统之类的，或者是旁路进化。这个就是游戏策划的事了。如果后续有大把时间。会在上边花些心思。**
